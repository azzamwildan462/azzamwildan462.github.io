Robot Operating System

Robot Operating System is a common Framework to develop a Robotics core. It has many benefits such as multithreading, fast, easy to communicate with websocket, and many more. I use ROS Melodic and Noetic to develop Robotics core and make a simulation with ROS-Gazebo and Docker.
